## Setup
Instalando gems

$ ```bundle install```

Criar o banco e as tabelas

$ ```rake db:create```

$ ```rake db:migrate```

Colocar alguns exemplos no banco :)

$ ```rake db:seed```

Rodar o Server

$ ```rails s```

ou

$ ```rails server```

Visualizar rotas:

$ ```rails routes```
